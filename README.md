# AI-Powered Scam & Phishing Detection

## Problem
Users frequently fall victim to scam job offers, phishing emails, malicious links, and fraudulent messages.

## Solution
An AI-based system using NLP and Machine Learning to analyze messages and URLs in real time and provide risk alerts before user interaction.

## Tech Stack
Python, Machine Learning, NLP, Flask, React, PostgreSQL.

## Status
Prototype stage - architecture and concept designed.
